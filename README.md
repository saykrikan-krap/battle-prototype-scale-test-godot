## Godot binaries (manual download)

These are the official Godot 4.5.1 stable downloads used for local runs.

- Linux x86_64: https://github.com/godotengine/godot/releases/download/4.5.1-stable/Godot_v4.5.1-stable_linux.x86_64.zip
- Windows x64: https://github.com/godotengine/godot/releases/download/4.5.1-stable/Godot_v4.5.1-stable_win64.exe.zip
- macOS universal: https://github.com/godotengine/godot/releases/download/4.5.1-stable/Godot_v4.5.1-stable_macos.universal.zip

We keep local binaries under `tools/godot/` and ignore them in git.

## Running

From the project root:

- macOS/Linux (bash/zsh): `./run.sh`
- Windows (PowerShell): `.\run.ps1`

Headless resolve (no rendering, `--resolve-only` auto-adds `--headless`):
- `./run.sh --resolve-only`
- `.\run.ps1 --resolve-only`

Resolve profiling (prints timing breakdown):
- `./run.sh --resolve-only --profile-resolve`
- `.\run.ps1 --resolve-only --profile-resolve`

Custom battle setup (manual placement UI):
- `./run.sh --custom-setup`
- `.\run.ps1 --custom-setup`

Map selection flags (default: trees):
- Clear map: `--map-clear`
- Tree map: `--map-trees`
- River map (water + 3-tile gap): `--map-river`

## Placeholder sprites

Unit placeholder sprites are generated by the script in `tools/generate_unit_sprites.gd`.

- macOS/Linux: `./run.sh --headless --script res://tools/generate_unit_sprites.gd`
- Windows (PowerShell): `.\run.ps1 --headless --script res://tools/generate_unit_sprites.gd`
